# CTRL + Q Writer Pro

An authentic, distortion-free online IELTS Writing practice canvas built to simulate strict examination conditions. Designed with an ultra-clean layout, this application ensures you practice relying entirely on your own skills—without the help of modern auto-correct features.

## 🔥 Key Features

* **Strict Exam Simulation:** Natively blocks spelling error highlights, grammar squiggles, and smart auto-correct indicators across mobile and desktop browsers.
* **Real-Time Telemetry:** Live counter tracking words, total characters, and estimated standard IELTS page counts dynamically as you type.
* **Instant PDF Archiving:** A single-click clean layout compiler that lets you download your written essay as a beautifully structured PDF document for grading or offline review.
* **Mobile-First Responsive Layout:** A sleek, minimal light-themed dashboard engineered to scale beautifully on any modern smartphone or tablet.
* **Community Driven:** Seamless native integration to join the **CTRL + Q** ecosystem via Discord.

---

## 🛠️ Tech Stack

* **Frontend:** Semantic HTML5, CSS3
* **Styling Framework:** Tailwind CSS (via CDN)
* **Logic/Telemetry:** Vanilla JavaScript (No external tracking or bloated dependencies)
